# Plan – NutriMind (MVP)

## Objective
Build a lightweight AI-powered food analysis web app focusing on UX and perceived intelligence.

## Scope
- Single-page application
- Input: user food description
- Output: AI-generated insights
- No persistent backend

## Tech Stack
- Frontend: React (Vite)
- Styling: Tailwind CSS
- AI: Prompt-based (mock or API)
- Deployment: GCP Firebase Hosting

## Constraints
- Time-bound (2 hours)
- Minimal dependencies
- No authentication
- No database

## Success Criteria
- Clean UI
- Smooth interaction
- Meaningful AI-like output
- Deployed and working link
