# 🧠 NutriMind – AI-Powered Food & Health Assistant

NutriMind is a lightweight, AI-inspired web application that analyzes daily food intake and provides actionable health insights. Built for a prompt-driven hackathon, it focuses on delivering intelligent feedback through a clean, modern UI without heavy backend dependencies.

---

## 🚀 Live Demo

👉 https://your-project.web.app

---

## ✨ Features

* 🥗 **Food Analysis**
  Enter your meal and get instant nutritional insights.

* 📊 **Macro Breakdown**
  Visual representation of protein, carbs, and fats.

* ⚡ **Health Rating**
  Classifies meals as Good / Moderate / Poor.

* 🧠 **AI Insights**
  Contextual explanation of nutritional quality.

* 💡 **Smart Suggestions**
  Actionable recommendations based on your meal.

* 🔄 **Smart Swap Suggestions**
  Suggests healthier alternatives (e.g., brown rice instead of white rice).

* 📜 **History Tracking**
  View previous food analyses.

* 📈 **Insights Dashboard**
  Displays deeper health optimization tips.

* 👤 **Profile Controls**

  * Reset Data
  * Basic session controls

---

## 🏗️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Charts:** Recharts (or equivalent)
* **AI Logic:** Prompt-based / Mocked intelligence
* **Deployment:** Google Cloud (Firebase Hosting)

---

## 🧠 How It Works

1. User inputs food + quantity
2. Input is processed via lightweight logic
3. Nutritional data is estimated (mock/logic-based)
4. AI-style insights and suggestions are generated
5. Results are visualized through UI components

## ⚙️ Installation & Setup

```bash

# Install dependencies
```
